Mobile Phone Market Analysis 📊

Overview
Analyzed the 2025 Mobile Phone Dataset using AWS PartyRock AI data analysis tool.
Submitted as Project 2 for AWS AI Practitioner Challenge — Udacity x AWS (April 2026).

Key Insights
- Mid-range phones ($300-$600) have better cameras than flagships
- Realme and Infinix offer best value for money
- Poco leads battery capacity with 10,000 mAh average
- Apple has worst value score despite $1,028 average price
- 19 brands analyzed with 1000+ records

Tools Used
- AWS PartyRock Data Analysis
- Excel, CSV

Dataset
- Source: Kaggle — Mobile Dataset 2025
- Records: 1000+
- Brands: 19
